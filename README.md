# HANGAKOBO: STOMACH CLOCK

おさのなおこの版画で作られた「木版画の灯」シリーズ。
シンメトリーで動く構造とスクロール演出を主体とした SPA です。  
Vanilla JS によるスクラッチで構成されており、画面全体を使った動的な演出を試みています。

---

## 特徴

- シンメトリーで動くスクロールとアニメーションによる演出多数
- シングルページ構成（SPA）
- HTML / CSS / JS のみで制作
- PC表示に最適化
